# Dashboard

A simple dashboard webapp 

## Installation

Use the [docker-compose](https://docs.docker.com/compose/) tool to build and run the dashboard.

```bash
git clone https://github.com/ngeorge-peillex/Dashboard.git
cd Dashboard
docker-compose --build -d
```
